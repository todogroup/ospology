## General Guidelines

- **Issues**: Use the [issue templates](.github/ISSUE_TEMPLATE/) to report bugs, suggest content, or propose translations
- **Pull Requests**: Fork the repo, create a branch, and submit a PR. Please keep PRs focused on a single change
- **Translations**: We welcome translations! Check existing efforts in `ospo-book/content/` before starting

## Code of Conduct

This project follows the [Linux Foundation Code of Conduct](https://www.linuxfoundation.org/code-of-conduct/).

To report security concerns, please email security@todogroup.org.
